# webface.cloud app template

The starting point for an app on [webface.cloud](https://webface.cloud) — a static frontend wired to a managed backend (database, auth, file storage, realtime) that already exists at the same origin.

## Get going (3 steps)

1. **Use this template** (green button above) to make your repo.
2. On your webface.cloud dashboard, open your app's card → **Frontend → Connect & deploy** → paste your repo URL. Copy the webhook URL it gives you into your repo's Settings → Webhooks (content type JSON).
3. Push. Your app redeploys itself on every push.

Point your coding agent at `AGENTS.md` (OpenCode and most harnesses read it automatically; Claude Code reads `CLAUDE.md`, same content). It explains the platform so the agent builds correctly on the first try.
